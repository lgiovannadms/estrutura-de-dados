# Estrutura de Dados e Algoritmos
Repositório criado para armazenar e organizar os códigos, algoritmos e conceitos estudados durante a disciplina de **Estrutura de Dados e Algoritmos** no curso de Ciência da Computação na UFCG.  O objetivo deste espaço é consolidar o aprendizado teórico por meio da prática e servir como portfólio pessoal.
